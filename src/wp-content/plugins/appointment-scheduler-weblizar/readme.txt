Appointment Scheduler is one stop solution for online bookings on WordPress. This plugin manages all for you.

== Description ==

Book appointments on your WordPress blog site.

Add **[appointment]** shortcode on any Page / Post to display booking form on website. The user can select services, date and time according to his need then fill his details and confirm the appointment for booking.

== Installation ==

1. Upload the entire **appointment-scheduler-weblizar** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin through the **Plugins** menu in WordPress admin.
3. Use **[appointment]** shortcode to show booking Form Page and Post.
